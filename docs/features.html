🏠 Overview

Code Agent is a comprehensive AI-powered development environment that combines the power of large language models with a rich set of development tools. It provides an intelligent CLI interface for software development tasks including code generation, analysis, debugging, testing, and project management.

💻

Terminal Execution

Execute any shell command with intelligent output parsing and error handling

📄

File Operations

Read, write, edit, and manage files with smart diff preview and undo support

🔍

Code Search

Powerful regex-based search across entire codebases with symbol indexing

🛠

Git Integration

Complete Git workflow support including commits, branches, and PR management

🤖

Multi-Agent System

Specialized agents for code review, debugging, testing, and documentation

🐘

Memory System

Persistent memory that learns from conversations and project context

⚙

Auto-Fix Errors

Intelligent error analysis with automatic fix suggestions for multiple languages

📦

Project Templates

Create new projects from templates for Python, Node.js, React, Go, Rust and more

Quick Start

# Start the agent
start.bat

# Or with Python directly
python -m src.cli

# Create a new project
/new myproject D:\projects python-api

# Ask the agent anything
"Create a REST API with user authentication"
